Common Requirements

Regardless of which path you choose, you'll need:

  • Service Account JSON Key — Downloaded from GCP Console
  • GCP Region — Where resources will be deployed (e.g., europe-west1)
  • Enabled APIs — Terraform enables these automatically
GCP Layer Limitations

GCP supports Layers 1-3 only. There are no managed Digital Twin (L4) or Grafana (L5) equivalents in GCP. For full L4/L5 support, use AWS or Azure.
