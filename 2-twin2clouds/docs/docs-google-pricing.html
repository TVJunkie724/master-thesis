Google Cloud Pricing Schema

Field sources, fetched dimensions, static defaults, and calculated logic for pricing.json (Google Cloud section).

Note: Google Cloud pricing is currently static. Dynamic fetching via the Google Cloud Billing Catalog API is planned for a future release.

Services & Fields

Data Transfer (Egress)

Service: Network

Field Value Notes
transfer.pricing_tiers Static Tiered pricing (First 100GB free/cheap, then tiered).
egressPrice $0.12 Base egress price.

Cloud IoT Core (Layer 1)

Service: Cloud IoT Core (Deprecated/Legacy)

Field Value Notes
pricePerMessage $0.0005 Per MB (approximate).
pricePerDeviceAndMonth $0 No monthly device fee.

Cloud Functions (Layer 2)

Service: Cloud Functions

Field Value Notes
requestPrice $0.40 Per million invocations.
durationPrice $0.0000025 Per GB-second.
freeRequests 2,000,000 Free tier invocations.
